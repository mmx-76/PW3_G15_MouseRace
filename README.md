PW3 Mouse Race Controller

Control firmware for the autonomous line-following mouse used in the University of Bath Engineering Practice & Design (EE22005) Project Week 3.

The system uses inductive sensing, closed-loop control, and differential motor drive to autonomously follow a track defined by a 20 kHz (kilohertz – one thousand cycles per second) alternating current signal.

The aim of the project is to design a mouse capable of completing a 20 m race track in the shortest possible time while maintaining stable track following.

System Overview

The mouse uses inductive sensing coils to detect the magnetic field generated by a wire embedded in the track. The difference between the signals from the left and right coils is used to estimate the mouse’s position relative to the track centre.

A microcontroller processes the sensor signals and adjusts the speed of the two drive motors using pulse width modulation (PWM – switching a digital signal rapidly to control average power).

Steering is achieved using differential drive, where the left and right motors run at slightly different speeds.

Software Modules

The firmware is structured into modular components.

1. Sensor Inputs

Reads the analogue values from the inductive coils and determines track offset.

2. Accelerometer Interface

Communicates with the MMA8451 accelerometer using I²C to measure acceleration and detect orientation changes.

3. Start Logic

The mouse remains idle until the start button is pressed.

4. Motor Control

Motor speeds are controlled using high-frequency PWM (~40 kHz) to reduce audible noise and improve motor driver efficiency.

5. Control Algorithm

The difference between coil signals is used to generate a steering correction which adjusts the relative motor speeds.

Development Notes

The project is developed incrementally using the following stages:

Sensor input validation

Accelerometer integration

Motor PWM control

Steering control implementation

Track-following optimisation

Testing is performed on progressively larger tracks before attempting the final race track.
